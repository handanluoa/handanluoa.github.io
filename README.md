这个页面已经基于当前目录中的文档重新整理，包含：

- 项目目标与核心问题
- 数据层级与 aesthetic buckets
- 三层文本监督结构
- metadata schema 示例
- 数据生产、审核与偏好学习工作流
- MVP 与后续路线

## 站点文件

- `index.html`
- `styles.css`
